Mosaic Creator
Repository
Mosaic Creator allows you to create mosaics with Java image processing. The program has a texture folder and an image input. Running the program will result in a new image which is created from the textures in the texture folder. It also automatically resizes images which are outside a desired range. You can also tell the program to only use certain textures that allow for the creation of very cool images.
Digi-Dig
Repository
Digi-Dig is a prototype for a future full game. It is a grid-based topdown shooter game about mining for ores. It features complex world generation and enemy AI. It was created in Unity using C#.
